# Roadmap to Production

Документ описывает необходимые доработки для перевода текущей версии Gantt AI из MVP-состояния в production-ready систему.

---

# Текущее состояние

На текущем этапе реализован MVP:

- интерактивная Gantt-диаграмма;
- создание и редактирование задач;
- работа с исполнителями;
- импорт и экспорт Excel;
- AI-ассистент с передачей контекста проекта;
- Docker-окружение для запуска.

Основная цель текущей версии — проверить архитектуру и возможность использования AI для управления проектами.

---

# Технический долг

## 1. Отсутствие постоянного хранения данных

### Текущее состояние

Данные проекта хранятся в оперативной памяти приложения.

После перезапуска сервера данные теряются.

### Что необходимо сделать

Добавить базу данных:

- PostgreSQL;
- ORM слой (SQLAlchemy);
- миграции Alembic.

Создать модели:

- User;
- Project;
- Task;
- TaskDependency;
- ProjectHistory.

---

## 2. Нет системы пользователей

### Текущее состояние

Приложение работает без авторизации.

### Что необходимо сделать

Добавить:

- регистрацию пользователей;
- авторизацию;
- JWT access/refresh токены;
- разграничение доступа к проектам.

---

## 3. Отсутствие истории изменений

### Текущее состояние

Изменения задач не сохраняются.

### Что необходимо сделать

Добавить аудит:

- кто изменил задачу;
- когда было изменение;
- какие поля были изменены.
